pangu.js ES Modules Demo

⚠️ This demo requires a web server!

ES modules cannot be loaded from file:// URLs due to CORS policy.

To run this demo:

# IMPORTANT: Run from the project root directory, not the examples folder!

# Option 1: Using Python
cd /path/to/pangu.js
python -m http.server 8000

# Option 2: Using Node.js
cd /path/to/pangu.js
npx http-server

# Then visit: http://localhost:8000/examples/demo-esm.html

ES Modules Import Example

This demo shows how to import and use pangu.js with ES modules:

import { pangu, BrowserPangu } from '../dist/browser/pangu.js';

// Use the default instance
const spaced = pangu.spacingText('你好world');

// Or create a new instance
const myPangu = new BrowserPangu();
myPangu.spacingPage();
